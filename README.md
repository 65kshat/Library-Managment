# Library Management System (Python + MySQL)

A simple command-line Library Management System built using Python and MySQL.  
This program allows users to store, view, and search books in a database.  
It also includes voice feedback using text-to-speech.

---

## Features

- Add new books to the library
- View all books stored in the database
- Search books by title
- Text-to-speech feedback
- Clean table display using tabulate
- MySQL database storage

---

## Technologies Used

- Python
- MySQL
- mysql-connector-python
- pyttsx3
- tabulate
